Consola - Sistema de incidentes
Desarrollo de una aplicación por consola para gestionar incidentes, utilizando Java y la tecnología JPA Hibernate para una eficiente interacción con bases de datos. El sistema permite realizar operaciones como agregar, visualizar, modificar y eliminar información de incidentes, técnicos, diagnósticos, soluciones, clientes y servicios. Utilicé patrones de diseño como State para manejar los estados de los incidentes (abierto, en preparación, cerrado) y Singleton para asegurar una única instancia de ciertos componentes. Además, implementé funciones lambda para generar reportes detallados, como rendimiento por técnico y análisis diarios y mensuales.
